# Vehicle-Sales-Projections-EXCEL-
Project Overview
This project demonstrates the complete end-to-end process of data cleaning and dashboard creation in Microsoft Excel. By taking raw customer data, we perform essential data preparation steps—such as removing duplicates and standardizing values—to create an interactive dashboard that visualizes purchasing trends.

Key Objectives
Data Cleaning: Transforming raw data into a structured and readable format (e.g., converting 'M'/'S' to 'Married'/'Single').
Data Transformation: Creating new calculated columns using nested IF statements to group continuous variables like Age into meaningful Age Brackets.
Dashboard Construction: Building interactive Pivot Tables and Charts to analyze customer demographics.
Interactivity: Implementing Slicers for dynamic filtering by Marital Status, Region, and Education.

Analysis Questions for Beginners
To further explore this dataset, try answering the following questions:

Demographics: Which age group shows the highest conversion rate for purchasing a vehicle?
Income Analysis: Is there a significant difference in the average income of individuals who purchased a vehicle compared to those who did not?
Commute Habits: Do customers with shorter or longer commutes show a higher preference for purchasing a vehicle?
Education Impact: How does the education level of a customer correlate with their likelihood of owning a car or purchasing a vehicle?
Regional Trends: Which region has the highest number of total vehicle sales, and how does the marital status distribution differ in that region?
